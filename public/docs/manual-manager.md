## 1. 시스템 개요

### 1.1 담당자 역할
담당자는 방문 신청서를 검토하고 승인/반려 처리를 담당합니다. 승인된 방문자에게는 QR 출입증이 자동 발급됩니다.

### 1.2 접속 정보
- **URL**: https://v0-lng-tml.vercel.app/admin
- **계정**: 관리자로부터 발급받은 ID/PW 사용

### 1.3 처리 흐름
```
신청서 접수 → SMS 알림 수신 → 신청서 검토 → 승인/반려 → 방문자에게 SMS 발송
```

## 2. 로그인 및 대시보드

### 2.1 로그인
1. 관리자 페이지 접속 (/admin)
2. 이메일/비밀번호 입력
3. **"로그인"** 버튼 클릭

### 2.2 대시보드 화면

#### 상단 통계 카드
| 카드 | 설명 |
|------|------|
| **대기중 신청** | 검토 대기 중인 신청 건수 |
| **승인완료** | 승인된 신청 건수 |
| **반려됨** | 반려된 신청 건수 |
| **취소됨** | 취소된 신청 건수 |

#### 월별 신청 현황
- 최근 6개월간 신청 추이 그래프
- 월 선택 시 해당 월 상세 통계 표시

#### 월별 상세 현황
- 선택한 월의 승인/반려/대기 건수
- 기본값: 현재 월

## 3. 신청서 관리

### 3.1 신청 목록 화면
**메뉴**: 대시보드 → 신청 목록 (또는 좌측 메뉴)

#### 목록 컬럼
| 컬럼 | 설명 |
|------|------|
| 접수번호 | 고유 접수번호 |
| 신청자 | 신청자 이름 |
| 소속 | 신청자 소속 |
| 방문 목적 | 방문 사유 요약 |
| 방문 기간 | 시작일 ~ 종료일 |
| 상태 | 대기중/승인/반려/취소 |
| 담당자확인 | 담당자 확인 체크박스 |

#### 필터 기능
- **상태별**: 전체/대기중/승인/반려/취소
- **날짜별**: 날짜 범위 선택
- **검색**: 신청자명, 소속, 접수번호 검색

### 3.2 상세보기

#### 열기 방법
- 목록에서 **행 클릭** → 상세 모달 열림

#### 상세 정보 확인
1. **신청 정보**: 방문 목적, 기간, 지역, 담당자
2. **신청자 정보**: 이름, 연락처, 생년월일, 소속, 직위
3. **동행인 정보**: 동행인 목록 및 상세 정보
4. **전자기기**: 반입 전자기기 목록
5. **첨부파일**: 항만이수증 등 첨부파일 확인/다운로드

### 3.3 승인 처리

#### 승인 방법
1. 상세보기 모달에서 **"승인"** 버튼 클릭
2. 확인 팝업에서 **"확인"** 클릭
3. 승인 완료 → 방문자에게 SMS 자동 발송

#### 승인 시 처리 사항
- QR 출입증 자동 생성 (신청자 + 동행인 각각)
- 방문자에게 SMS 발송 (QR 출입증 링크 포함)
- 상태가 "승인완료"로 변경

### 3.4 반려 처리

#### 반려 방법
1. 상세보기 모달에서 **"반려"** 버튼 클릭
2. **반려 사유** 입력 (필수)
3. **"확인"** 클릭

#### 반려 시 처리 사항
- 상태가 "반려됨"으로 변경
- 방문자에게 반려 SMS 발송
- 반려 사유가 방문자 조회 화면에 표시됨

### 3.5 담당자 확인 체크

#### 기능
- 본인이 담당자인 신청서를 **확인했음** 표시
- 목록에서 담당자확인 체크박스 클릭

#### 시각적 구분
- **본인이 담당자인 경우**: 해당 셀이 노란색(앰버) 배경으로 강조

## 4. 출입 현황 조회

### 4.1 정문 출입현황
**메뉴**: 출입현황 → 정문 출입현황

#### 조회 기능
- **날짜 선택**: 좌우 화살표 또는 달력으로 날짜 이동
- **범위 검색**: 시작일 ~ 종료일 범위 조회
- **새로고침**: 실시간 데이터 갱신

#### 표시 정보
| 항목 | 설명 |
|------|------|
| 방문자명 | 입장한 방문자 이름 |
| 소속 | 방문자 소속 |
| 입장 시간 | QR 스캔 시간 |
| 퇴장 시간 | 퇴장 스캔 시간 (있는 경우) |
| 첨부파일 | 항만이수증 확인 가능 |

### 4.2 부두 출입현황
**메뉴**: 출입현황 → 부두 출입현황

#### 부두 선택
- **1부두** / **2부두** 탭으로 전환
- 각 부두별 출입 이력 조회

### 4.3 첨부파일 확인
- 방문자명 클릭 시 첨부파일(항만이수증) 확인 가능
- **신청자**와 **동행인**의 첨부파일이 각각 구분되어 표시
- 첨부파일이 없는 경우 "없음"으로 표시

## 5. SMS 알림

### 5.1 자동 발송 SMS

#### 신청 접수 시
- **수신자**: 담당자
- **내용**: 새로운 방문 신청 알림

#### 승인 시
- **수신자**: 신청자 + 동행인 (각각)
- **내용**: 승인 안내 + QR 출입증 링크

#### 반려 시
- **수신자**: 신청자
- **내용**: 반려 안내 + 반려 사유

### 5.2 SMS 발송 오류
- **"발신번호 미등록"** 오류: SOLAPI 대시보드에서 발신번호 등록 필요
- SMS 발송 실패 시에도 승인/반려 처리는 정상 완료됨
