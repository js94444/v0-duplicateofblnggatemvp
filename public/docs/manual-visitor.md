## 1. 시스템 개요

### 1.1 출입관리시스템란?
보령LNG터미널 방문자 출입 관리 시스템입니다. 방문 신청부터 승인, QR코드 출입증 발급, 현장 출입까지 모든 과정을 온라인으로 처리할 수 있습니다.

### 1.2 접속 방법
- **URL**: https://v0-lng-tml.vercel.app
- **지원 브라우저**: Chrome, Safari, Edge (최신 버전 권장)
- **모바일**: 스마트폰 브라우저에서도 이용 가능

### 1.3 방문 신청 흐름
```
방문 신청 작성 → 담당자 검토 → 승인/반려 → QR 출입증 발급 → 현장 출입
```

## 2. 방문 신청하기

### 2.1 메인 화면에서 시작
1. 홈페이지 접속
2. **"방문 신청하기"** 버튼 클릭

### 2.2 신청 유형 선택
- **일반 방문**: 회의, 미팅, 견학 등 일반적인 방문
- **작업 방문**: 공사, 정비, 설치 등 작업 목적의 방문

### 2.3 신청 정보 입력

#### 방문 정보
| 항목 | 설명 | 필수 |
|------|------|------|
| 방문 목적 | 방문 사유를 상세히 기재 | O |
| 방문 기간 | 시작일 ~ 종료일 선택 | O |
| 방문 지역 | 방문할 구역 선택 | O |
| 담당자 선택 | 방문 담당자를 선택 (검색 가능) | O |

#### 신청자 정보
| 항목 | 설명 | 필수 |
|------|------|------|
| 이름 | 신청자 본인 이름 | O |
| 연락처 | SMS 수신용 휴대폰 번호 | O |
| 생년월일 | YYYY-MM-DD 형식 | O |
| 소속 | 회사/기관명 | O |
| 직위 | 직급/직책 | O |

#### 동행인 정보 (선택)
- **"동행인을 추가하시려면 클릭하세요"** 버튼으로 추가
- 동행인별로 이름, 연락처, 생년월일, 소속, 직위 입력
- 동행인도 개별 QR 출입증 발급 가능

#### 전자기기 반입 (선택)
- **스마트폰 외** 전자기기 반입 시에만 입력
- 기기명, 모델명, 시리얼번호, 반입 사유 기재
- 신청자/동행인 각각 별도로 등록 가능

#### 첨부파일
- **항만보안교육 이수증**: 필요 시 업로드
- 지원 형식: PNG, JPG, PDF
- 최대 파일 크기: 10MB

### 2.4 신청 완료
1. 모든 정보 입력 후 **"신청하기"** 버튼 클릭
2. **접수번호** 확인 및 저장 (중요!)
3. 담당자에게 SMS 알림 자동 발송
4. 승인 시 신청자에게 SMS로 QR 출입증 링크 발송

## 3. 출입증 상태 조회

### 3.1 조회 방법
1. 메인 화면에서 **"출입증 조회하기"** 클릭
2. **접수번호** 입력 후 조회

### 3.2 상태 종류
| 상태 | 설명 |
|------|------|
| **대기중** | 담당자 검토 대기 중 |
| **승인완료** | 승인 완료, QR 출입증 발급됨 |
| **반려** | 신청 반려됨 (사유 확인 가능) |
| **취소** | 신청자가 취소함 |

### 3.3 조회 화면 정보
- 신청 정보 요약
- 방문 기간 및 방문 지역
- 동행인 목록
- QR코드 출입증 (승인 시)
- 반려 사유 (반려 시)

## 4. QR코드 출입증 사용

### 4.1 QR코드 수신
- 승인 완료 시 **SMS로 QR 출입증 링크** 발송
- 링크 클릭하여 QR코드 화면 확인

### 4.2 출입 방법
1. 출입구(정문/부두)에서 QR코드 화면 제시
2. 경비원이 **QR 스캐너**로 스캔
3. **입장 허용/거부** 화면 확인
4. 퇴장 시에도 동일하게 QR 스캔

### 4.3 주의사항
- QR코드는 **방문 기간 내**에만 유효
- **본인 확인** 절차가 있을 수 있음
- 동행인은 **각자의 QR코드** 사용
- 스마트폰 화면 밝기를 높여주세요

## 6. 자주 묻는 질문

### Q1. 접수번호를 잊어버렸어요.
> 담당자에게 문의하시거나, SMS 기록을 확인해주세요.

### Q2. 승인이 언제 되나요?
> 담당자 검토 후 승인되며, 승인 시 SMS로 알림이 발송됩니다.

### Q3. 방문 일정이 변경되었어요.
> 기존 신청을 취소하고 새로 신청해주세요.

### Q4. 동행인도 별도 QR코드가 필요한가요?
> 네, 동행인도 각자의 QR코드로 출입합니다.

### Q5. QR코드가 스캔되지 않아요.
> 화면 밝기를 높이고, QR코드가 깨끗하게 보이도록 해주세요. 문제 지속 시 경비실에 문의하세요.
