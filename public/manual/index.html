담당자가 확인하고 내부 승인이 완료되면, 방문자와 동행인에게 QR 출입증이 문자로 자동 발송됩니다. 입장도, 퇴장도 QR 한 번이면 끝납니다.

01

방문 신청

방문자가 홈페이지에서 3가지 서약에 동의한 뒤, 방문 정보·차량·동행인·반입 기기 등을 입력하고 신청합니다.

02

담당자 통보 및 확인

신청 접수 즉시 담당자에게 SMS 알림이 발송됩니다. 담당자가 신청서를 검토하고 확인 의견(승인/반려)을 남깁니다.

03

내부 승인

담당자 확인 후 최종 승인합니다. 신청자와 동행인 각각에게 QR 출입증 링크가 자동 발송됩니다.

04

QR 스캔으로 현장 출입

경비 초소에서 QR코드를 스캔합니다. 녹색 화면이 나오면 입장 가능입니다.

해당 가이드가 나옵니다

방문하시는 분은 방문자 가이드만 보시면 신청부터 현장 출입까지 순서대로 따라 할 수 있습니다. 사내 담당자는 담당자 탭을 확인하세요.

01

홈페이지 접속 후 "방문신청" 클릭

visit.lng-tml.com에 접속합니다. 메인 화면에서 "방문신청" 카드를 누릅니다.

상시출입증 소유자는 별도의 방문 신청 없이 출입할 수 있습니다. 단, 부두지역(제1·2부두) 출입 시에는 상시출입증 소유와 관계없이 반드시 사전 방문 신청이 필요합니다. Chrome, Safari, Edge 최신 버전을 권장하며 스마트폰에서도 이용할 수 있습니다.
02

서약 동의 (3가지 필수)

개인정보 수집·이용 동의(3년 보관), 보안 서약, 안전준수 서약에 모두 동의해야 신청서 작성으로 진행됩니다. "전체 동의" 후 "신청서 작성"을 누르세요.

03

기본 정보 입력

이름·휴대전화번호·생년월일·소속·직책·회사주소를 입력합니다(이메일은 선택). 차량을 가져오면 차량번호·유종(휘발유/경유/LPG/전기)·불꽃방지망 보유 여부(Y/N)를 입력하고, 차량이 없으면 "차량없음"을 체크합니다.

스마트폰 외 전자기기(노트북·태블릿 등)를 반입하려면 "전자기기 추가"로 품명·모델명·시리얼넘버·반입 사유를 등록합니다.
⚠️항만(부두) 출입자는 항만이수증 첨부가 필수입니다. 제1·2부두 등 부두지역에 출입하는 경우, 신청서에 항만안전교육 이수증을 반드시 첨부해야 합니다. (교육 이수: 항만안전교육 포털 kptiedu.kr)
단, 일반지역·공정지역만 출입하는 경우에는 항만이수증이 필요하지 않습니다.
04

방문 정보 입력

담당자(BLT 내부 담당자 이름을 정확히 입력해 검색·선택), 방문 목적(업무협의·공사/작업·물품반입출·점검/감사·부두작업·견학·기타 중 선택), 출입지역(일반지역/공정지역/제1부두/제2부두), 상세 방문 사유, 방문 시작일·종료일을 설정합니다.

방문 기간은 최대 14일까지 신청할 수 있습니다. 출입지역 옆 "출입지역약도"로 구역을 확인하세요. (일반지역=일반만 / 공정지역=일반+공정 / 제1·2부두=일반+공정+선택 부두)
05

동행인 등록 (선택)

동행인이 있으면 "동행인을 추가하시려면 클릭"으로 이름·휴대전화·생년월일·소속·직책을 입력합니다. 동행인마다 개인정보·보안·안전준수 동의를 각각 체크해야 하며, 필요 시 동행인별 전자기기·항만이수증도 등록합니다.

동행인은 별도 차량을 이용할 수 없습니다. 차량을 가져오는 사람은 각자 따로 방문 신청을 해주세요.
06

"신청하기" 후 접수번호 저장

신청 완료 후 화면에 접수번호가 표시됩니다. 반드시 저장해 두세요 — 상태 확인 및 QR 코드에 사용됩니다.

07

신청 상태 확인 · 변경 · 취소

메인 화면 "신청현황"에서 휴대전화번호로 검색하여 모든 신청서 상태를 확인할 수 있습니다. 상세 화면에서 승인 전(대기중)에는 "변경" 버튼으로 직접 수정할 수 있고, 대기중·승인 상태 모두 "방문취소"가 가능합니다.

승인대기: 변경/취소 가능 (수정 시 재심사 대기로 전환) | 승인완료: QR 코드 확인 가능, 취소 가능 | 신청반려: 반려 사유 확인 가능
08

승인 시 QR 코드 수신 및 현장 출입

승인 완료 시 SMS로 QR 코드 링크가 발송됩니다(동행인은 각자 수신). 링크로 QR 화면을 열어 경비 초소에 제시하며, 화면 밝기를 최대로 하면 스캔이 더 잘 됩니다. 신청현황 상세 화면에서도 QR을 다시 열 수 있습니다.

✅ 입장 허용녹색 화면 + 체크 아이콘이 표시되면 입장 가능합니다.
❌ 입장 거부빨간 화면이 나오면 사유를 확인하고 담당자에게 문의하세요.
09

기타 기능

안전보건환경안내서(SHE): 메인 화면에서 안전/보건/환경 안내 자료 확인 | 게시판: 건의·문의 작성

항만안전교육은 항만안전교육 포털 kptiedu.kr에서 이수합니다.
01

관리자 페이지 로그인

visit.lng-tml.com/admin/login에 접속합니다.

02

대시보드 확인

로그인 후 대시보드에서 방문 현황을 한눈에 파악합니다. 전체/대기/승인/반려/취소 건수, 월별 트렌드, 신청 유형 분포, 상위 방문 기관 순위를 확인할 수 있습니다.

03

신청 관리 — 본인 담당 건 조회

신청 관리 메뉴에서 자신이 담당자로 지정된 신청만 조회됩니다. 접수번호·담당자명·조직명·방문자명으로 검색하고, 상태·출입지역·날짜 범위로 필터링할 수 있습니다.

담당자는 자신이 담당자로 지정된 신청만 볼 수 있습니다. 전체 조회는 특수경비대 또는 보안담당자만 가능합니다.
04

담당자 확인 의견 등록

신청서 내용을 확인한 뒤 "담당자 확인"에서 승인 / 반려 의견을 선택하고 필요 시 메모를 남깁니다. 이 의견은 내부 승인을 돕는 참고용이며, 최종 승인/반려는 보안담당자가 처리합니다.

담당자에게는 최종 승인·반려 권한이 없습니다. 담당자 확인 의견은 모든 관리자에게 공유되어 보안담당자가 최종 결정 시 참고합니다.
05

방문 캘린더

월별 캘린더 뷰에서 방문 일정을 확인합니다. 신청 유형별 색상 구분(단체방문=보라, 항만출입=청록, 개인방문=에메랄드), 상태 배지, 이벤트 클릭 시 상세 팝업을 제공합니다.

01

관리자 로그인 및 전체 신청 조회

visit.lng-tml.com/admin/login에 접속하여 보안 계정으로 로그인합니다. 대시보드 및 캘린더를 확인하고, 신청 관리에서 모든 신청서를 조회할 수 있습니다.

담당자와 달리 특수경비대는 담당자 필터 없이 모든 신청서를 조회할 수 있습니다.
02

출입현황 모니터링

출입현황 메뉴에서 탭(정문 / 부두)을 전환하며 실시간 출입 상황을 모니터링합니다. 부두 탭은 다시 1부두/2부두로 나뉩니다. 대기 · 체크인(현재 체류) · 체크아웃(퇴장 완료) · 재입장 수를 요약 카드로 확인합니다.

수동 체크인/아웃, 재입장 처리, 카드번호 일괄 등록, 항만이수증 열람, 날짜/상태 필터, 이름 검색 기능을 활용하세요. 최근 활동은 하이라이팅됩니다.
03

QR 스캐너로 게이트 출입 처리

visit.lng-tml.com/scanner에 접속해 스캐너 인증(PIN)을 거친 뒤, 인증 방식(QR/휴대폰번호), 방향(입장/퇴장), 게이트(정문/1부두/2부두)를 설정합니다. 카메라로 QR 코드를 스캔하면 즉시 결과가 표시됩니다.

🟢 입장 허용 (ALLOW)녹색 화면 + 체크 아이콘. 방문자명·소속·방문 기간 확인 후 입장 안내. 잠시 후 자동 복귀.
🔴 입장 거부 (DENY)빨간 화면 + X 아이콘. 거부 사유 확인 → 담당자 연락 안내. 노란색(MISMATCH)은 중복 스캔.
04

부두 출입 관리

부두 출입현황 화면에서 방문자의 카드번호, 이름, 생년월일을 확인합니다. 카드번호를 입력해 일괄 저장한 뒤, 해당 행의 "체크인" 또는 "체크아웃" 버튼으로 출입을 처리합니다. (부두는 정문 통과 여부와 무관하게 독립적으로 운영됩니다.)

재입장 처리: 퇴장 완료된 방문자가 재입장해야 할 경우, 해당 인원의 체크박스를 선택한 뒤 "재입장" 버튼을 클릭합니다. 새로운 행이 생성되며 체크인 시간이 자동으로 기록됩니다.
05

QR 스캔 안 될 때 대처

카메라가 작동하지 않으면 인증 방식을 "휴대폰번호"로 전환하여 처리합니다. 당일 승인된 신청서를 자동으로 조회합니다.

QR 분실 시: 방문자 폰에서 신청현황 접속 → 휴대전화번호로 조회 → 접수번호 클릭 → QR 재확인 가능.
06

근무시간 외 승인·반려

공휴일 및 근무시간 외에는 특수경비대가 보안담당자를 대신하여 신청 관리에서 승인·반려를 수행할 수 있습니다. 승인 시 신청자·동행인에게 QR이 자동 발송되며, 반려 시 사유를 입력해야 합니다.

근무시간 내(평일 09:00~18:00)에는 보안담당자가 전담합니다. 모든 승인·반려 이력은 시스템에 기록됩니다.
07

긴급 상황 대처

시스템 장애 시 수기로 방문자 정보(이름·소속·시간)를 기록하고 대외협력팀에 즉시 연락합니다. 시스템 복구 후 수기 기록을 입력합니다.

📞 문의: 대외협력팀 041-939-9955 | 보안 연락처로 지정된 경우 새 방문 신청 접수 시 SMS 알림을 자동 수신합니다.
01

보안담당자 로그인 — 모든 기능 접근

visit.lng-tml.com/admin/login에 접속합니다. 보안담당자는 담당자+특수경비대의 모든 기능(대시보드, 신청 관리, 캘린더, 출입현황, 게시판, 스캐너)과 계정 관리에 접근할 수 있습니다.

02

신청 승인 처리

신청 관리에서 대기중(PENDING) 또는 검토중(UNDER_REVIEW) 신청서를 선택하고 "승인" 버튼을 클릭합니다. 승인 시 승인 의견(선택)을 남길 수 있습니다.

승인 시 자동 수행: 신청자 및 동행인별 QR 패스 생성 → 신청자에게 QR 코드 링크 SMS 발송 → 동행인에게 개별 SMS → 담당자에게 승인 알림 SMS → (이메일 입력 시) 이메일 알림 발송
03

신청 반려 처리

해당 신청서에서 "반려" 버튼을 클릭하고 반려 사유를 입력합니다 (필수). 신청자와 담당자에게 반려 사유가 포함된 SMS 및 (이메일 입력 시) 이메일이 자동 발송됩니다.

근무시간 내(평일 09:00~18:00)에는 보안담당자가 전담합니다. 공휴일·근무시간 외에는 특수경비대도 승인/반려가 가능합니다. 담당자가 남긴 확인 의견을 참고하세요.
04

승인 취소

이미 승인한 신청을 되돌려야 할 경우 승인 취소를 실행합니다. 신청 상태가 접수 대기로 복귀하고, 발급된 QR 패스는 무효화(REVOKED)되며, 신청자와 담당자에게 취소 안내 SMS가 발송됩니다.

05

계정 관리

계정 관리 메뉴에서 관리자 계정을 생성·수정·삭제합니다. 아이디(사번), 이름, 연락처, 비밀번호, 역할(보안담당자 / 특수경비대 / 담당자)을 설정합니다.

비밀번호 초기화 시 임시 비밀번호가 발급되며, 해당 계정은 첫 로그인 시 변경이 강제됩니다. CSV 파일로 대량 계정 일괄 등록(username·name·role·password 컬럼)도 가능합니다. 본인 계정은 삭제할 수 없습니다.
06

역할 권한 관리

계정 관리 → 권한 탭에서 특수경비대·담당자 역할이 접근할 수 있는 페이지(대시보드, 신청 관리, 방문 캘린더, 출입현황, 게시판)를 역할별로 켜고 끌 수 있습니다.

보안담당자의 권한은 항상 전체 접근이며 변경할 수 없습니다. 계정 관리 페이지는 보안담당자 전용입니다.
07

보안 연락처 관리

계정 관리 → 보안 연락처 탭에서 보안담당자 계정을 보안 연락처로 지정하고 전화번호를 설정합니다. 새 방문 신청이 접수될 때마다 지정된 연락처로 SMS가 자동 발송됩니다.

08

개인정보 보유기간 관리

대시보드에서 수집·이용 동의 후 보유기간(3년)이 경과한 개인정보 건수를 확인하고, 해당 데이터를 마스킹 처리할 수 있습니다. (보안담당자 전용)

마스킹은 되돌릴 수 없으므로 실행 전 안내 문구를 반드시 확인하세요.

아직 승인 전(승인대기)이라면 신청현황 상세 화면의 "변경" 버튼으로 직접 수정할 수 있습니다. 수정하면 다시 검토 대기 상태가 됩니다. 이미 승인된 경우에는 "방문취소" 후 새로 신청하거나 담당자에게 직접 연락하세요.
상시출입증 소유자는 일반적으로 별도의 방문 신청 없이 출입할 수 있습니다. 다만 부두지역(제1·2부두)에 출입할 때는 상시출입증 소유 여부와 관계없이 반드시 사전 방문 신청이 필요합니다.
네, 동행인은 각자의 QR코드로 출입합니다. 신청 시 동행인 정보와 동의 항목을 입력하면 승인 후 동행인 번호로 개별 QR 출입증이 발송됩니다. 동행인은 별도 차량을 이용할 수 없으니, 차량 이용자는 따로 신청해 주세요.
담당자 확인 의견은 참고용이며, 보안담당자의 최종 승인이 있어야 방문자에게 QR이 발송됩니다. 최종 승인까지 완료됐는데 SMS가 안 왔다면 발신번호 등록 등 SMS 발송 설정을 확인하세요.
출입지역에 따라 다릅니다. 제1·2부두 등 부두(항만) 지역에 출입하는 경우에는 항만안전교육 이수증 첨부가 필수입니다. 신청서에 이수증을 첨부하지 않으면 부두 출입 승인이 어려울 수 있습니다. 반면 일반지역이나 공정지역만 출입하는 경우에는 항만이수증이 필요하지 않습니다. 항만안전교육은 항만안전교육 포털(kptiedu.kr)에서 이수할 수 있습니다.
공휴일 및 근무시간 외(평일 18:00~익일 09:00, 주말/공휴일)에는 특수경비대가 슈퍼어드민을 대신하여 승인·반려를 수행할 수 있습니다. 근무시간 내(평일 09:00~18:00)에는 슈퍼어드민만 가능합니다. 승인/반려 이력은 모두 시스템에 기록됩니다.
담당자는 본인이 담당자로 지정된 신청만 조회하고, 확인 의견(승인/반려)과 메모를 남길 수 있습니다(최종 권한 없음). 슈퍼어드민은 모든 신청 조회, 최종 승인·반려, 승인 취소, 계정 관리, 역할 권한 설정, 보안 연락처 관리, 개인정보 보유기간 관리 등 시스템 전체를 관리합니다.
