01

방문 신청

방문자가 홈페이지에서 방문 정보, 동행인, 반입 기기 등을 입력하고 신청합니다.

02

담당자 SMS 수신 및 검토

신청 접수 즉시 담당자에게 SMS 알림이 발송됩니다. 담당자가 신청서를 검토합니다.

03

보안담당자 최종 승인

담당자 확인 후 보안담당자가 최종 승인합니다. 방문자에게 QR 출입증 링크가 자동 발송됩니다.

04

QR 스캔으로 현장 출입

경비 초소에서 QR코드를 스캔합니다. 녹색 화면이 나오면 입장 가능입니다.

01

홈페이지 접속 후 "방문 신청하기" 클릭

v0-lng-tml.vercel.app에 접속합니다. 메인 화면에서 "방문 신청하기" 버튼을 누릅니다.

Chrome, Safari, Edge 최신 버전을 권장합니다. 스마트폰에서도 이용할 수 있습니다.
02

서약 동의 (3가지 필수)

개인정보 처리방침(3년 보관), 보안 서약서, 안전 준수 서약서에 모두 동의해야 다음 단계로 진행됩니다.

03

신청 정보 입력

이름·연락처·생년월일·소속·직위를 입력합니다. 담당자 선택, 방문 목적, 출입 구역, 방문 기간(최대 14일)을 설정합니다.

차량 정보, 전자기기(노트북·태블릿 등 모델/시리얼/용도) 신고, 동반 방문자 등록도 가능합니다. 동반자에게도 개별 QR 출입증이 발급됩니다. 항만 구역 접근 시 안전교육 수료증 업로드가 필요합니다.
04

"신청하기" 후 접수번호 저장

신청 완료 후 화면에 접수번호가 표시됩니다. 반드시 저장해 두세요 — 상태 확인 및 QR 코드에 사용됩니다.

05

신청 상태 확인

메인 화면 "신청 현황 조회"에서 휴대전화번호로 검색하여 모든 신청서 상태를 확인할 수 있습니다.

대기중: 수정/취소 가능 | 검토중: 담당자 확인 중 | 승인됨: QR 코드 확인 가능 | 반려됨: 반려 사유 확인 가능
06

승인 시 QR 코드 수신 및 현장 출입

승인 완료 시 SMS로 QR 코드 링크가 발송됩니다. SMS 링크로 QR 화면을 열고 경비 초소에 제시합니다. 화면 밝기를 최대로 설정하면 스캔이 더 잘 됩니다.

✅ 입장 허용녹색 화면 + 체크 아이콘이 표시되면 입장 가능합니다.
❌ 입장 거부빨간 화면이 나오면 사유를 확인하고 담당자에게 문의하세요.
07

기타 기능

SHE 가이드라인: 안전/보건/환경 안내 자료 다운로드 | 오시는 길: 본사 주소, 카카오맵/네이버맵 연동 | 건의사항 게시판: 건의·버그 리포트·문의 작성 가능

01

관리자 페이지 로그인

v0-lng-tml.vercel.app/admin/login에 접속합니다. 슈퍼어드민이 생성한 계정으로 로그인합니다.

최초 로그인 시 반드시 비밀번호를 변경해야 합니다.
02

대시보드 확인

로그인 후 대시보드에서 방문 현황을 한눈에 파악합니다. 전체/대기/승인/반려/취소 건수, 월별 트렌드(6개월), 신청 유형 분포, 상위 8개 방문 기관 순위를 확인할 수 있습니다.

03

신청 관리 — 본인 담당 건 조회

신청 관리 메뉴에서 자신이 담당자로 지정된 신청만 조회됩니다. 접수번호·담당자명·조직명·방문자명으로 검색하고, 상태·출입 구역·날짜 범위로 필터링할 수 있습니다.

매니저는 자신이 담당자(contact_name)로 지정된 신청만 볼 수 있습니다. 전체 조회는 경비대 또는 슈퍼어드민만 가능합니다.
04

검토 확인 (체크박스)

신청서를 열어 내용 확인 후 "담당자확인" 체크박스를 클릭합니다. 이 확인은 "검토 완료" 표시이며, 승인/반려와는 다릅니다. 이후 슈퍼어드민이 최종 승인하면 방문자에게 QR 출입증이 자동 발송됩니다.

매니저는 승인 또는 반려 권한이 없습니다. 승인/반려는 슈퍼어드민(또는 공휴일·근무시간 외 특수경비대)만 가능합니다.
05

방문 캘린더

월별 캘린더 뷰에서 방문 일정을 확인합니다. 신청 유형별 색상 구분(단체=보라, 항만=청록, 개인=에메랄드), 상태 배지, 이벤트 클릭 시 상세 팝업을 제공합니다.

06

매니저가 할 수 없는 것

신청 승인/반려, 계정 관리, 역할 권한 수정은 불가합니다. QR 출입 현황 조회는 슈퍼어드민이 별도 권한을 부여한 경우에만 가능합니다.

01

관리자 로그인 및 전체 신청 조회

v0-lng-tml.vercel.app/admin/login에 접속하여 보안 계정으로 로그인합니다. 대시보드 및 캘린더를 확인하고, 신청 관리에서 모든 신청서를 조회할 수 있습니다.

매니저와 달리 경비대는 담당자 필터 없이 모든 신청서를 조회할 수 있습니다.
02

QR 출입 현황 모니터링

QR 출입 현황 메뉴에서 탭별 구역(정문/부두1/부두2)을 전환하며 실시간 출입 상황을 모니터링합니다. 대기 방문, 현재 체크인, 체크아웃 완료, 총 승인 수를 요약 카드로 확인합니다.

수동 체크인/아웃, 재입장 추적, 카드번호 일괄 등록, 항만 수료증 열람, 날짜/상태 필터, 이름 검색 기능을 활용하세요. 최근 10분 활동은 하이라이팅됩니다.
03

QR 스캐너로 게이트 출입 처리

v0-lng-tml.vercel.app/scanner에서 인증 방식(QR/전화번호), 방향(입장/퇴장), 게이트(정문/1부두/2부두)를 설정합니다. 카메라로 QR 코드를 스캔하면 즉시 결과가 표시됩니다.

🟢 입장 허용 (ALLOW)녹색 화면 + 체크 아이콘. 방문자명·소속·방문 기간 확인 후 입장 안내. 3초 후 자동 복귀.
🔴 입장 거부 (DENY)빨간 화면 + X 아이콘. 거부 사유 확인 → 담당자 연락 안내. 노란색(MISMATCH)은 중복 스캔.
04

부두 출입 관리

부두 출입현황 화면에서 방문자의 카드번호, 이름, 생년월일을 확인합니다. 확인 후 해당 카드 우측 상단의 "체크인" 또는 "체크아웃" 버튼을 클릭하여 출입을 처리합니다.

재입장 처리: 퇴장 완료된 방문자가 재입장해야 할 경우, 해당 인원의 체크박스를 선택한 뒤 "재입장" 버튼을 클릭합니다. 새로운 행이 생성되며 체크인 시간이 자동으로 기록됩니다.
05

QR 스캔 안 될 때 대처

카메라가 작동하지 않으면 인증 방식을 "휴대전화번호"로 전환하여 처리합니다. 당일 승인된 신청서를 자동으로 조회합니다.

QR 분실 시: 방문자 폰에서 신청현황 접속 → 휴대전화번호로 조회 → 접수번호 클릭 → QR 재확인 가능.
06

⚡ 공휴일/근무시간 외 — 신청 승인·반려

공휴일 및 근무시간 외(평일 18:00~익일 09:00, 주말/공휴일 전일)에는 슈퍼어드민을 대신하여 방문 신청의 승인·반려를 수행할 수 있습니다.

터미널은 24시간 운영되지만 슈퍼어드민이 부재할 수 있는 시간대에 긴급 방문을 처리하기 위한 위임 권한입니다.

승인 절차: 신청 관리 → 대기중/검토중 신청서 선택 → 상세 내용 확인 → "승인" 버튼 클릭 → QR 패스 자동 생성 + 방문자·동반자·담당자에게 SMS/이메일 자동 발송

반려 절차: 해당 신청서 선택 → "반려" 버튼 클릭 → 반려 사유 입력(필수) → 방문자에게 반려 사유 포함 SMS/이메일 자동 발송

07

승인·반려 시 유의사항

승인 전 확인: 방문 목적 타당성, 출입 구역 적절성, 항만 구역 시 안전교육 수료증 여부, 동반자 정보 완전성, 전자기기 신고 내용. 반려 사유는 구체적으로 작성하여 재신청 시 참고하도록 합니다.

승인/반려 이력은 시스템에 기록되며 슈퍼어드민이 추후 확인 가능합니다. 판단이 어려운 경우 슈퍼어드민에게 유선 연락 후 처리를 권장합니다.
08

긴급 상황 대처

시스템 장애 시 수기로 방문자 정보(이름·소속·시간)를 기록하고 IT 관리팀에 즉시 연락합니다. 시스템 복구 후 수기 기록을 입력합니다.

📞 안전 문의: 대외협력팀 041-939-9955 | 보안 연락처로 지정된 경우 새 방문 신청 접수 시 SMS 알림을 자동 수신합니다.
01

슈퍼어드민 로그인 — 모든 기능 접근

v0-lng-tml.vercel.app/admin/login에 접속합니다. 슈퍼어드민은 매니저+경비대의 모든 기능(대시보드, 캘린더, 전체 신청 조회, QR 출입 현황, 스캐너)에 접근할 수 있습니다.

02

신청 승인 처리

신청 관리에서 대기중(PENDING) 또는 검토중(UNDER_REVIEW) 신청서를 선택하고 "승인" 버튼을 클릭합니다.

승인 시 자동 수행: 방문자 및 동반자별 QR 패스 생성 → 방문자에게 QR 코드 링크 SMS 발송 → 동반 방문자에게 개별 SMS → 담당자에게 SMS → 이메일 알림 발송
03

신청 반려 처리

해당 신청서에서 "반려" 버튼을 클릭하고 반려 사유를 입력합니다 (필수). 방문자에게 반려 사유가 포함된 SMS 및 이메일이 자동 발송됩니다.

근무시간 내(평일 09:00~18:00)에는 슈퍼어드민이 전담합니다. 공휴일·근무시간 외에는 특수경비대도 승인/반려가 가능합니다.
04

계정 관리

계정 관리 메뉴에서 관리자 계정을 생성·수정·삭제합니다. 아이디, 이름, 연락처, 비밀번호, 역할(super_admin / security / manager)을 설정합니다.

비밀번호 초기화 시 임시 비밀번호가 발급되며, 해당 계정은 첫 로그인 시 변경이 강제됩니다. CSV 파일로 대량 계정 일괄 등록도 가능합니다. 본인 계정은 삭제할 수 없습니다.
05

역할 권한 관리

계정 관리 → 권한 탭에서 security와 manager 역할이 접근할 수 있는 페이지(대시보드, 신청 관리, 캘린더, QR 출입 현황)를 동적으로 설정합니다.

슈퍼어드민의 권한은 항상 전체 접근이며 변경할 수 없습니다. 계정 관리 페이지는 슈퍼어드민 전용입니다.
06

보안 연락처 관리

계정 관리 → 보안 연락처 탭에서 슈퍼어드민 계정을 보안 연락처로 지정하고 전화번호를 설정합니다. 새 방문 신청이 접수될 때마다 지정된 연락처로 SMS가 자동 발송됩니다.

07
